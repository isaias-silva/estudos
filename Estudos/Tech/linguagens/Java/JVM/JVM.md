máquina virtual java que garante a execução de programas Java.

- interpreta bytecode gerado pelo javac(.class)
- gerencia a memória(Garbage collector);




>  o arquivo gerado pela jvm chama-se jar, e pode ser executado em qualquer ambiente que possua o [[JRE]].

> Para desenvolver com java em um ambiente é necessário utilizar o kit de desenvolvimento [[JDK]]
