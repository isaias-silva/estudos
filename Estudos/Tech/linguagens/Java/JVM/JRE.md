ambiente de execução java, necessário para execução de programas java no sistema.

**JRE** é a junção da[[JVM]] com as principais classes padrões do java:

- `java.lang`
- `java.util`
- `java.io`
- `java.net`


o JRE junto do compilador java e ferramentas de desenvolvimento formam o kit de desenvolvimento [[JDK]].
