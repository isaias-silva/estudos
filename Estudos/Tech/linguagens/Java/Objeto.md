Um objeto é a instância de uma [[Classe]], permitindo que seus membros de instância sejam utilizados.

## Instanciação:

```java 

Alarm alarm = new Alarm("2025-01-10");

alarm.start();//start é um membro de instância sendo possível usa-lo apenas após instanciar um objeto da classe Alarm.


```


## métodos presentes em todos os objetos.

<font color="#2DC26B">.getClass()</font>: retorna o nome da classe que o objeto instancia

<font color="#2DC26B">.equals()</font>: compara objetos.

<font color="#2DC26B">.toString()</font>: converte objeto para String.
