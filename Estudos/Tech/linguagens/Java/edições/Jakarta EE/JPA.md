**Java Persistence Api**, especificação padrão java para trabalhar com banco de dados utilizando objetos.

JPA permite o mapeamento de classes Java para tabelas de bancos de dados(Object Relational Mapping - ORM).

Basicamente é a ponte entre java e banco de dados sem a necessidade de escrever queries SQL.

### principais conceitos:

##### Entity
  Classe que representa uma tabela

##### EntityManager
   Geerencia entidades(tabelas).
   - salva
   - busca
   - remove
   - atualiza.

#### implementações: [^1]
- Hibernate ORM
- EclipseLink



[^1]: JPA é apenas uma especificação, seu uso se dá pelas suas implementações.
