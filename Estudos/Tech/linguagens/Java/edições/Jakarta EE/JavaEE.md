Java Enterprise edition, plataforma java voltada para aplicações web e corporativas.

> Hoje é chamado de Jakarta EE [^1]

o JavaEE utiliza o [[JavaSE ]]como base e traz soluções prontas para diversos problemas em aplicações web:

 - [[servlet]]
 - [[JSP]]
 - [[JPA]]
 - [[CDI]]
 - [[JTA]]
 - [[JMS]]
 - [[JAX-RS]]
 - [[JAX-WS]]
 - [[Security]]


[^1]: java EE foi renomeado para JarkataEE, assim como seus pacote, anteriormente conhecido como Javax, foi substituído após o java EE 8 por Jakarta.
